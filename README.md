# Neko Calculator — Scientific Calculator with Tkinter

Neko Calculator adalah aplikasi **scientific calculator desktop** berbasis Python & Tkinter dengan tampilan modern (Dark/Light mode), fitur matematika lanjutan, keyboard support, serta penyimpanan riwayat perhitungan.
Project ini dibuat untuk menunjukkan kemampuan Python, GUI development, dan clean code practices.

#Fitur Utama
**Basic & Scientific Calculator**
  - Operasi dasar: `+ - × ÷`
  - Scientific: `sin`, `cos`, `√ (sqrt)`
  - Menggunakan derajat (degree) untuk fungsi trigonometri

**Dark Mode & Light Mode**
  - Toggle tema dengan satu klik
  - Desain konsisten & nyaman dilihat

**Keyboard Support**
  - Angka & operator langsung dari keyboard
  - `Enter` untuk hitung
  - `Backspace` untuk hapus
  - `Esc` untuk clear

**History Perhitungan**
  - Riwayat hasil tampil otomatis
  - Scrollable history panel

**Copy to Clipboard**
  - Salin hasil perhitungan dengan satu klik

**Save History**
  - Simpan riwayat perhitungan ke file `.txt`
